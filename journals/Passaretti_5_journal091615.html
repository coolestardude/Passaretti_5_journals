~the ol and ul tag mean ordered and unordered list. li seperates the words within ul and ol by either adding a bullet or number

~the tr tag is used to define a row. td seperates the lines within the tr tag for a table

~the a tag is used to link to another page

~the aside tag defines some content aside from the content it is placed in. (HTML5 semantic element: predefined CSS element:what to expect inside the tags)

~the footer tag makes a section at the page underneath the body

~margin-left and margin-right show how much empty space is to the right and left of the body

~width edits how wide a specified thing is(like a div)

~height edits how tall a specified thing is

~background-color changes what color the bg is of a specified text

~color changes the color of the actual text

~a:hover will modify "a" when the mouse hovers over it(can also use :link, :visited, :active)

~text-decotation modifies the font and size of the text

~span means nothing by itself. It is used to group things(NON semantic:opposite of aside)

--new semantic elements: details, article, aside, figcaption,figure, footer, header, main, mark, nav, section, summary, and time.
